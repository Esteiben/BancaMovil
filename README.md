# Diseño Responsivo del Inicio de Sesión - App Banco Pichincha

## Descripción del Proyecto

Este proyecto consiste en el desarrollo de una interfaz de inicio de sesión responsiva para una aplicación móvil inspirada en la App del Banco Pichincha, realizada en Android Studio utilizando Material Design 3.

El objetivo principal es recrear una experiencia visual moderna, limpia y adaptable a diferentes tamaños de pantalla, aplicando principios de diseño responsivo y buenas prácticas de desarrollo móvil en Android.

# Objetivos

- Diseñar una interfaz moderna similar a una aplicación bancaria real.
- Aplicar diseño responsivo para distintos dispositivos Android.
- Implementar tarjetas/botones interactivos con iconografía moderna.
- Mejorar la experiencia visual y la organización de la interfaz.


# Tecnologías Utilizadas

- Android Studio
- Java / XML
- ConstraintLayout
- Vector Drawables (Material Icons)


# Características Implementadas

## Pantalla de Inicio de Sesión

La interfaz contiene:

- Logo principal de la aplicación
- Campos de acceso
- Botones interactivos tipo tarjeta
- Diseño limpio y minimalista
- Estilo inspirado en aplicaciones bancarias modernas
